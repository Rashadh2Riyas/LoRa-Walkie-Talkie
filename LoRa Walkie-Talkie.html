LoRa Walkie-Talkie Guide

Build your own LoRa-based walkie-talkie for under 20,000 LKR with this guide!

Step-by-Step Guide

  1. Gather all the components listed in the guide.
  2. Follow the pin connection instructions to wire the components.
  3. Upload the Arduino code provided to your Arduino UNO.
  4. Test the setup by pressing the PTT button and observing transmission and reception.
  5. Assemble the components into an enclosure.

Pin Connections

LoRa SX1278 to Arduino UNO

LoRa SX1278 PinArduino UNO PinDescription
VCC3.3VPower (3.3V supply)
GNDGNDGround
NSS (CS)D10Chip Select
SCKD13Serial Clock
MISOD12Master-In-Slave-Out
MOSID11Master-Out-Slave-In
RESETD9Reset Pin
DIO0D2Interrupt (optional)

Microphone to A0 (Analog pin for reading audio signals). Speaker to D6 using a simple amplifier circuit.
